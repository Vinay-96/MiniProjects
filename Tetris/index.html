How to play

  1. Click on the Start/Pause Button to Start or Pause
  2. Use the keyBoard arrow keys to move
  3. left key to move left
  4. Right key to move right
  5. Up key to change the shape
  6. Down key to move faster
  7. If score becomes end the its gameover
  8. Refresh the page to start the new game
  9. Repeat the above steps and have fun..!
